# نظام تدقيق وتحليل الملفات وإعداد التقارير

نظام محلي/ويب خفيف لتحليل ملفات بيانات متغيرة. يتيح للمستخدم رفع ملفات، تحديد الأعمدة المطلوبة، وعرض الإحصائيات والرسوم البيانية والتقارير.

## تنبيه هام
هذا النظام غير مرتبط بأي اسم ملف ثابت (مثل gg.txt).

## طريقة التشغيل

### تشغيل الواجهة (Frontend)
```bash
cd frontend
npm run dev
```

### تشغيل الباكند (Backend)
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```
أو استخدام:
```bash
./run_backend.ps1
```

## مراحل التطوير القادمة
- دعم رفع الملفات والمعاينة.
- إضافة مكتبات Recharts و TanStack Table للواجهة.
- ربط الفلاتر والإحصائيات.
- إضافة ميزات التصدير المتعدد.
